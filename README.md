#  Steam AI Game Recommender (VEBPR)

A high-performance video game recommendation system for Steam, powered by the **VEBPR (View-Enhanced Bayesian Personalized Ranking)** algorithm. This project includes an optimized Cython training engine and a real-time Streamlit web interface.

##  Key Features
*   **Advanced Algorithm:** Implements VEBPR, which outperforms standard BPR by leveraging both interaction (playtime) and auxiliary (view/metadata) signals.
*   **Real-time Inference:** Uses "Folding-In" technology to calculate personalized recommendations for new users in milliseconds without retraining the entire model.
*   **Interactive UI:** Built with Streamlit, allowing users to select their favorite games and see instant recommendations with cover art.
*   **High Performance:** Features a custom Cython-based SGD engine with interaction-based sampling for fast and robust training on large datasets (300k+ users).

##  Tech Stack
*   **Language:** Python, Cython
*   **Libraries:** NumPy, Pandas, SciPy, Optuna (for hyperparameter tuning), Streamlit
*   **Optimization:** Custom 30-bit FastRand for unbiased sampling on Windows systems.

##  How It Works
1.  **Training:** The model learns item embeddings ($i_{factors}$) and biases ($i_{biases}$) by optimizing the BPR loss function on the Steam interaction dataset.
2.  **Folding-In:** When a new user selects games they like, the system performs a mini-batch SGD on the fly to estimate a unique user vector ($u_{factor}$) that aligns with their chosen games.
3.  **Ranking:** Recommendations are generated by calculating the dot product of the user vector and all item vectors, filtered to exclude games already owned by the user.

##  Performance
The VEBPR model achieved an **NDCG@10 of 0.0311**, significantly outperforming standard BPR (0.024) and popularity-based baselines.
